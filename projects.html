Suspicious Activity (Shoplifting) Detection using YOLOv8 Pose

Developed a robust system to detect suspicious activities, particularly shoplifting, by leveraging YOLOv8 for precise pose estimation. This system extracts human poses from video frames and uses an XGBoost classifier for action recognition, effectively distinguishing between normal and suspicious behaviors.

To enhance usability, I integrated an alert feature that sends an email notification to the user whenever the model's confidence level exceeds 0.95, ensuring timely and actionable alerts. This project showcases my expertise in computer vision, machine learning, and real-time alert systems, contributing to enhanced security measures in retail environments.

AutoML - Automated EDA and modeling
